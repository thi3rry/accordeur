# accordeur
Accordeur de guitare

Vibe codé avec Chat GPT. 

Pourquoi ?
-> Marre des applications native bourrée de tracker, un accodeur a uniquement besoin du flux audio d'un micro pour te donner une indication visuelle de correspondance entre une note et la fréquence audio.
Simple et ça doit le rester.


Disponible sur https://thi3rry.github.io/accordeur

PWA offline.

Sans pub ni tracker.


## Développement

```shell
python -m http.server 8080 -d public
```
